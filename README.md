# DriverKnowledgeTest
This is a program for the knowledge test of drivers in Alberta.

## Requirements
- Python 3.9
- PIL (install it by ```pip install pillow```)
- (And ... that's it!)

## Run
```
python doTest.py
```

## Adjust Arguments
Change ```NUM_OF_QUESTION_PER_SET``` to adjust the number of questions in a set. (30 by default, maximum 108)
